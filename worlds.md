# Worlds Guide

Keystone has multiple worlds, each with a different purpose. Choose based on your playstyle.

## World Overview

| World | Purpose | PVP | Difficulty | Resets |
|-------|---------|-----|------------|--------|
| Spawn | Hub area | No | - | Never |
| Survival | Building & living | Optional | Easy | Never |
| Resource | Mining & gathering | Yes | Easy | Weekly |
| PVP Arena | Factions & raiding | Yes | Hard | Never |
| Nether | Nether dimension | Yes | Hard | Never |
| The End | End dimension | Yes | Hard | Never |

## Spawn (Hub)

The central hub where all players start.

### Features
- Portal access to all worlds
- Tutorial signs and info boards
- Starter kit access
- Warp point (`/spawn`)

### Getting There
```
/spawn
```

### Protection
- No PVP
- No building
- No mob spawning

---

## Survival World

The main world for building and peaceful gameplay.

### Purpose
- Build your base
- Start farms
- Long-term projects
- Safe storage

### Features
- **Permanent** - Builds stay forever
- **Land claims** - Protect your area
- **Easy difficulty** - Less mob damage
- **PVP optional** - Off by default in claims

### Getting There
```
/warp survival
```
Or walk through the **green (east) portal** at spawn.

### Best Jobs Here
- Farmer
- Builder
- Woodcutter

### Tips
1. Claim your land with GriefPrevention
2. Set a home (`/sethome`)
3. Build away from spawn for privacy
4. Use the return portal to get back

---

## Resource World

A mining-focused world that resets periodically.

### Purpose
- Mining ores
- Gathering materials
- Strip mining without guilt
- Getting rare resources

### Features
- **Resets weekly** - Fresh ores every reset
- **No permanent builds** - Don't build here
- **Full resources** - Untouched terrain
- **Easy difficulty** - Focus on mining

### Getting There
```
/warp resource
```
Or walk through the **cyan (north) portal** at spawn.

### Best Jobs Here
- Miner (primary)
- Hunter (cave mobs)

### What Happens on Reset?
- World terrain regenerates
- All placed blocks disappear
- Your inventory stays with you
- Ores and caves respawn

### Tips
1. **Don't build bases here** - They'll be deleted
2. Bring food and torches
3. Mine deep for diamonds
4. Return to spawn before logging off
5. Check announcements for reset schedule

### Ore Earnings (Miner Job)

| Ore | $KEY per Block |
|-----|----------------|
| Coal Ore | 0.10 |
| Iron Ore | 0.20 |
| Gold Ore | 0.50 |
| Diamond Ore | 3.00 |
| Ancient Debris | 20.00 |

## PVP World (Factions)

Full PVP, land claiming, and raiding. High risk, high reward.

### Purpose
- Factions gameplay
- PVP combat
- Raiding enemy bases
- Competitive play

### Features
- **Full PVP** - Kill or be killed
- **Factions** - Claim land with your team
- **Raiding** - Attack enemy territory
- **Hard difficulty** - Mobs hit harder
- **Drop items on death** - Keep inventory OFF

### Getting There
```
/warp pvp
```
Or walk through the **red (south) portal** at spawn.

### Best Jobs Here
- Raider (PVP kills pay $KEY)
- Hunter (dangerous mobs)

### Warning
⚠️ **You WILL lose items if you die**
⚠️ **Other players CAN and WILL attack you**
⚠️ **Your base CAN be raided**

Don't bring anything you can't afford to lose.

### Factions Basics

See the full [Factions Guide](factions.md) for details.

Quick commands:
```
/f create <name>     - Create faction
/f claim             - Claim land
/f invite <player>   - Invite member
/f home              - Teleport to faction home
```

## Nether

The vanilla Nether dimension.

### Accessing
Build a nether portal in Survival or PVP world, or use:
```
/warp nether
```

### Features
- Standard nether terrain
- Nether fortresses
- Bastions
- Ancient debris

### Best Jobs Here
- Miner (ancient debris, nether gold)
- Hunter (blazes, wither skeletons)

### Tips
1. Bring fire resistance potions
2. Watch for ghasts
3. Coordinates link to overworld (8:1 ratio)
4. Good for wither skeleton farming

---

## The End

The End dimension with dragon and end cities.

### Accessing
Find and activate an End portal, or after initial clear:
```
/warp end
```

### Features
- Ender Dragon boss
- End cities with loot
- Shulkers
- Elytra

### Best Jobs Here
- Hunter (endermen, dragon, shulkers)

### Tips
1. Dragon drops 200 KEY (Hunter job)
2. End cities have great loot
3. Shulkers drop shulker shells
4. Bring ender pearls for travel

---

## Traveling Between Worlds

### Portals (At Spawn)
Walk through the colored portals:
- Cyan → Resource
- Green → Survival
- Red → PVP
- Yellow → Market

### Warps
```
/warp spawn
/warp survival
/warp resource
/warp pvp
/warp market
```

### Return Portals
Each world has a blue return portal near its spawn point that takes you back to the main spawn.

### Homes
Set and use homes within worlds:
```
/sethome <name>
/home <name>
/homes         - List your homes
/delhome <name> - Delete a home
```

## World Rules

### All Worlds
- No cheating/hacking
- No exploiting bugs
- Report issues to staff

### Survival World
- Respect land claims
- No griefing unclaimed builds
- Keep chat friendly

### PVP World
- PVP is allowed everywhere
- Raiding is allowed
- Scamming in trades is NOT allowed
- Toxic behavior still prohibited

### Resource World
- Don't build permanent structures
- First come, first served on ores
- World resets - nothing is saved
